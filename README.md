# Penn Station Advocacy Campaign

A React-based advocacy website for Penn Station improvements and through-running implementation.

## Development

```bash
cd penn-station-advocacy
pnpm install
pnpm dev
```

## Build

```bash
cd penn-station-advocacy
pnpm run build
```
